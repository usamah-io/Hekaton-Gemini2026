### Fitur Utama SKS-Master

- **Kuis Cerdas Gemini:** Soal-soal pilihan ganda yang diracik instan oleh Gemini AI berdasarkan subjek dan tingkat kesulitan pilihanmu secara dinamis.
- **Pengecoh Jawaban Cerdas:** Pengecoh (*distractors*) dibuat dengan tingkat kesulitan akademis tinggi untuk mendeteksi kelemahan logika berpikirmu.
- **Pembahasan Komprehensif:** Pembahasan akademis menyeluruh untuk setiap soal kuis. Pahami akar permasalahan konsep secara mendalam dan cepat.
- **Tanpa Database & Cepat:** Kecepatan akses tanpa latensi query database. Sepenuhnya *serverless*, kencang, dan siap diakses kapan saja demi kebutuhan SKS.
- **Riwayat Progres Belajar:** Indikator skor ujian terakhir tersimpan aman di browser LocalStorage. Lacak kemampuan belajarmu tanpa database eksternal.
- **Ekspor Offline (.txt):** Ekspor daftar jawaban salah lengkap dengan pembahasan akademisnya agar dapat dipelajari secara offline di mana saja.
